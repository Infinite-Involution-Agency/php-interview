##  函数
1. `php` 中 `array_merge` 与 `+` 做数组连接的区别？
-  `array_merge` 对于 索引数组 会忽略 索引值 ，相同`key`只是将其合并在一起不会覆盖，对于关联数组则是，相同`key`后者覆盖前者的值
-  `+` 不管关联还是索引 都是以 前者为准，相同 key 值的后者会被忽略
```
$a = ["k"=>1, 222, 4];
$c = ["k"=>11, "1"=>22, 0=>33];

print_r(array_merge($a, $c));
Array
(
    [k] => 11
    [0] => 222
    [1] => 4
    [2] => 22
    [3] => 33
)

print_r(($a + $c));
Array
(
    [k] => 1
    [0] => 222
    [1] => 4
)

```
